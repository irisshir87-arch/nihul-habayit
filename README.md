# ניהול הבית — משפחת זילכה

גרסה 9: אפליקציית PWA משותפת לאיריס ולתומר, עם כניסה באמצעות סיסמה משפחתית בלבד במסך, שמירה ב-Supabase וסנכרון בזמן אמת.

## הפעלה מקומית

פתיחת `index.html` ללא הגדרת Supabase מפעילה מצב מקומי בדפדפן.

## חיבור לענן

יש לעדכן את `config.js` לפי `SETUP.md`, ולאחר מכן להעלות את כל הקבצים ל-GitHub Pages.
